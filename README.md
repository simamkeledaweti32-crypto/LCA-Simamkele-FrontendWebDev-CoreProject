## 1. Project Overview
This is a proof-of-concept, front-end only HR web application built for **ModernTech Solutions**, a software company in the healthcare industry. 

**Problem it solves:**  
ModernTech was using scattered Excel sheets + Word docs to manage employees, payroll, time-off, and performance reviews. This app centralizes everything into one responsive, user-friendly interface for HR staff.

## 2. Key Features Implemented
Based on the project brief requirements:
- **Employee Management**: Add, view, edit, delete employee records with personal info, salary, employment history. 15+ dummy records included.
- **Payroll + Digital Payslips**: Automated payroll calculation based on hours worked + salary rate. Generates downloadable payslips.
- **Time Off + Attendance Tracking**: Employees submit leave requests. HR approves/denies. Attendance is tracked and visualized.
- **Responsive Design**: Works on desktop, tablet, and mobile. Built with Bootstrap.
- **Client-side Storage**: Uses `localStorage` to simulate data persistence without backend.
- **Data Visualization**: Charts for attendance/reports using Chart.js [or your library].
- **Mock Authentication**: Hardcoded HR login to access the dashboard.

## 3. Tech Stack
